addRequest()method
public function addRequest(request:URLRequest, handler:Function, token:String = null, binary:Boolean = false):void

添加数据请求

Parameters
request:URLRequest — 请求
 
handler:Function — 请求处理函数

处理函数必须接收一个Object数据，比如 function requestHandler(data:Object):void; data为请求所得数据

 
token:String (default = null) — 请求标识

该token值用于集中处理请求结果。当请求中心请求成功/失败，将抛出事件RequestEvent.COMPLETE 或RequestEvent.FAILED，此时事件携带数据将包含该token值以便集中处理函数识别。

 
binary:Boolean (default = false) — 是否以二进制方式请求数据
