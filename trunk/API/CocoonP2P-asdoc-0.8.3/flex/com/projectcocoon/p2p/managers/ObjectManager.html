public function request(metadata:ObjectMetadataVO):MessageVO

Requests a shared object. Once requested, the object will be replicated. During replication, ObjectEvent.OBJECT_PROGRESS events get dispatched. When the object replication is finished, an ObjectEvent.OBJECT_COMPLETE event gets dispatched
